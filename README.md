# MultiDentNet-A-Unified-Deep-Learning-Framework-for-Automated-Diagnosis-of-Multiple-Dental-Conditions
MultiDentNet: A deep-learning model for automated diagnosis of 5 dental conditions (caries, gingivitis, discoloration, ulcers, hypodontia) from intraoral images. Achieves 99.37% accuracy with strong interpretability. First unified AI solution for multi-condition dental diagnostics.
